# Miko
```
//  ./.env

TOKEN=(bot token)
CLIENT_ID=(bot id)
DEV=(dev discord uid)
OSU_API_KEY=(osu! api v1 key)
OSU_CLIENT_ID=(osu! api v2 client id)
OSU_CLIENT_SECRET=(osu! api v2 client secret)

DROID_TRACKING_ENABLED=( true | false )
OSU_TRACKING_ENABLED=( true | false )
MONGO_URI=(mongodb uri)
```

```
$ git clone https://github.com/floemia/miko.git
$ cd ./miko
$ npm i
$ npm start
```
